### 手动验证清单

**完整流程测试**：
1. [ ] 提交视频生成任务
2. [ ] 立即刷新页面 → 任务继续生成（不消失）
3. [ ] 等待任务完成 → 右侧显示视频预览
4. [ ] 点击"查看历史" → 跳转到历史页面
5. [ ] 历史页面显示刚才的任务记录
6. [ ] 筛选"已完成" → 只显示 completed 的记录
7. [ ] 点击记录 → 弹窗显示详情（如果实现）
8. [ ] 点击下载 → 成功下载视频文件
9. [ ] 点击删除 → 记录从列表消失
10. [ ] 再次刷新页面 → 历史记录仍然存在

**异常场景测试**：
- [ ] 提交任务时数据库断开 → 显示错误提示（不崩溃）
- [ ] 刷新时数据库断开 → 从 localStorage 恢复（兜底）
- [ ] 删除不存在的记录 → 返回 404 错误
- [ ] 访问不存在的 task_id → 返回 404 错误
