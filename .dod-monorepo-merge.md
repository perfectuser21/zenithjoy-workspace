# DoD: ZenithJoy Monorepo 整合

## 验收清单

- [ ] `apps/geoai/` 存在，包含原 `geoai/` 的所有代码文件（package.json、src/、astro.config.mjs 等）
- [ ] `services/creator/` 存在，包含原 `creator/` 的所有代码文件（scripts/、requirements.txt 等）
- [ ] `services/jnsy-label/` 存在，包含原 `JNSY-Label/` 的所有文件（Dockerfile、app/ 等）
- [ ] 根 `package.json` workspaces 包含 `"apps/*"`（自动覆盖 apps/geoai）
- [ ] `.gitignore` 不会误忽略 services/ 或 apps/geoai/ 下的代码
- [ ] `git status` 显示新增文件正确，无多余文件
- [ ] CI 配置识别新项目结构（不破坏现有 CI）
- [ ] `workspace/` 重复 checkout 问题已记录（建议删除）

## 测试方式

```bash
# 验证目录结构
ls apps/geoai/package.json
ls services/creator/requirements.txt
ls services/jnsy-label/Dockerfile

# 验证 git 状态
git status --short | grep -E "^A"

# 验证 workspaces 配置
cat package.json | python3 -c "import sys,json; d=json.load(sys.stdin); print(d['workspaces'])"
```
