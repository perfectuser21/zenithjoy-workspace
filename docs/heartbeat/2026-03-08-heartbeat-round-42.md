### 问题 2：系统为什么仍在派发任务，尽管 Scheduler disabled？

**观察**：
```json
{
  "enabled": false,
  "loop_running": true,
  "last_tick": "2026-03-08T13:48:57.374Z",
  "last_dispatch": {
    "success": true,
    "task_id": "3dddd1b2-0c34-4131-98b1-7d01306ca16c",
    "dispatched_at": "2026-03-08T13:48:57.721Z"
  }
}
```

**解释**：
- `enabled: false` 意思是"不自动规划新任务"
- 但 `loop_running: true` 意思是"循环在执行现有队列中的任务"
- 当前 heartbeat 任务正是被循环派发的

**所以**：系统不是完全瘫痪，而是"规划瘫痪 + 执行继续"。
