## 技术细节

### Brain API 调用

**查询 KR 信息**：
```bash
curl -s "http://localhost:5221/api/brain/goals?dept=zenithjoy&type=area_okr" | jq '.[] | {id, title, parent_id, priority}'
```

**创建 Project**：
```bash
curl -X POST "http://localhost:5221/api/brain/goals" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Project Name",
    "description": "Detailed description",
    "parent_id": "kr_id",
    "priority": "P0",
    "type": "project",
    "status": "pending_decomposition"
  }'
```

**验证创建**：
```bash
curl -s "http://localhost:5221/api/brain/goals/{kr_id}/projects" | jq '.[]'
```
