# Task Card: ZenithJoy 内容工厂模块

## 需求（What & Why）

**功能描述**: 在 ZenithJoy 前端新增「内容工厂」模块，供员工查看内容生产排期、Pipeline 进度、审核待办。
内容类型三大类：视频 / 长文 / 图文，每类下有多个系列（目前只有 solo-company-case 属于长文）。

**核心页面**:
1. **生产看板** — Kanban 四列（调研中/生成中/待审核/已完成），按视频/长文/图文分组，每张卡片显示关键词+当前阶段+优先级
2. **排期日历** — 左栏日期列表，右栏当天内容详情，支持新建 Pipeline
3. **审核队列** — 需人工批准的内容列表，支持预览/通过/打回操作

**不做什么**: 不实现实际内容预览（占位即可），不改后端逻辑，不做权限细分。
