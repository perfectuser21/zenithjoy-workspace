### 1. CDP 连接测试

**脚本**：
```javascript
const playwright = require('playwright');

async function testCDP() {
  const browser = await playwright.chromium.connectOverCDP('http://localhost:19222');
  const contexts = browser.contexts();
  console.log('Contexts:', contexts.length);
  const context = contexts[0];
  const pages = context.pages();
  console.log('Pages:', pages.length);
  const page = pages[0];
  console.log('Current URL:', page.url());
}

testCDP();
```

## 关键坑点记录

### 从今日头条和抖音学到的教训

| 平台 | 坑点 | 解决方案 |
|------|------|---------|
| 今日头条 | 文章需要两步发布 | 拆分成两个脚本 |
| 抖音 | 文章封面虽显示可选但必需 | 强制要求封面 |
| 抖音 | 发布按钮需要 XPath | 使用精确的 XPath |
| 通用 | SSH 多行 JSON 传输失败 | 使用 Base64 编码 |
| 通用 | 文件上传需要等待 | 增加足够的等待时间 |
