# PRD: 修复 AI 视频生成页面白屏问题

## 问题描述
AI 视频生成页面点击"开始生成视频"按钮后，页面变成白屏，无法正常使用。

## 根本原因
1. 缺少平台抽象层（platforms/）实现
2. ToAPI 响应格式不一致（创建时直接返回，轮询时包装在 {code, data} 中）
3. 响应未正确解包导致 task.id 等字段为 undefined
4. React 组件尝试访问 undefined.slice() 导致崩溃

## 解决方案

### 1. 创建平台抽象层
- `src/api/platforms/base.ts` - 定义统一接口
- `src/api/platforms/toapi.ts` - ToAPI 适配器（包含响应解包逻辑）
- `src/api/platforms/index.ts` - 平台注册

### 2. 修复 ToAPI 响应解包
```typescript
// 在 getTaskStatus 中正确解包
if (responseData.code === 'success' && responseData.data) {
  task = responseData.data;  // 提取 data
} else {
  task = responseData;  // 直接使用
}
```

### 3. 更新 API 层
- 替换 `video-generation.api.ts` 使用平台抽象
- 统一参数格式：platform, model, prompt, duration, aspectRatio, resolution

## 验收标准
1. ✅ 页面不再白屏
2. ✅ 能成功提交视频生成请求
3. ✅ 任务状态正常轮询显示
4. ✅ 完成后显示视频预览

## 技术细节
- 平台：ToAPI (Google VEO3)
- 端点：https://toapis.com/v1
- 固定时长：8秒
- 宽高比：16:9 或 9:16
