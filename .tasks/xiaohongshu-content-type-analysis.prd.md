# 小红书图文 vs 视频字段分析 PRD

## 背景

已完成抖音和快手的图文 vs 视频字段分析，发现：
- **快手**：图文和视频字段完全相同，都有完播率
- **抖音**：图文和视频字段完全不同，图文无完播率，有专属的"文案展开率"和"平均浏览图片"

现在需要分析小红书平台的图文和视频内容类型差异。

## 目标

分析小红书平台的图文和视频作品，提取完整的字段清单，明确两者的区别。

## 产出物

1. **字段对比表**：
   - 视频完整字段列表（含字段名、示例值、说明）
   - 图文完整字段列表（含字段名、示例值、说明）
   - 标注专属字段和共同字段

2. **区分规则**：
   - 如何在页面/API中区分视频和图文
   - 关键标识字段（如抖音的"X张"、快手的时长）

3. **数据来源**：
   - 管理页面URL
   - 统计/分析页面URL
   - API端点（如果有）

4. **采集策略**：
   - 推荐的采集方式（API/DOM/其他）
   - 是否需要分别处理视频和图文
