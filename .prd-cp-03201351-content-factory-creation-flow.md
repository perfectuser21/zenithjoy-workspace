# PRD: 内容工厂创作链路

## 概述
打通内容工厂从前端到西安 Codex 的完整创作链路。员工在前端填写关键词、选择内容类型和目标平台后，点击执行，pipeline 自动在西安 Mac mini 上完成调研→生成→审核→导出四阶段，完成后前端展示卡片产出。

## 变更范围
1. `apps/dashboard/src/pages/ContentFactoryPage.tsx` — 前端
2. `packages/brain/src/routes/content-pipeline.js` — 后端 run API
3. `packages/brain/src/task-router.js` — 路由改 xian
4. `packages/brain/src/content-pipeline-executors.js` — executor 改 codex-bridge 派发

## 成功标准
1. [BEHAVIOR] 前端创建 Pipeline 后出现「开始执行」按钮，点击后触发执行
2. [BEHAVIOR] 执行过程中进度条自动刷新
3. [BEHAVIOR] Pipeline 完成后前端展示卡片图片预览
4. [ARTIFACT] 后端 POST /api/brain/pipelines/:id/run API 可用
5. [BEHAVIOR] executor 通过 codex-bridge 派发到西安
6. [GATE] 不破坏已有功能
