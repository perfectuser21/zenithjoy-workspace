## PRD - 快手图文发布脚本（CDP 自动化）

**需求来源**: Brain 自动调度，KR1 多平台发布扩展
**功能描述**: 实现快手创作平台的图文帖子自动发布，架构对标微博发布脚本，通过 CDP 控制 Windows PC 上已登录的 Chrome 浏览器完成发布
**涉及文件**:
- 新建 `workflows/platform-data/workflows/publisher/scripts/publish-kuaishou.cjs`
- 更新 `workflows/platform-data/workflows/publisher/scripts/README.md`
- 新建 `workflows/platform-data/workflows/publisher/scripts/test-kuaishou-publish.sh`

**成功标准**:
- 脚本接受 JSON 内容文件（含文案 + 图片路径列表）
- 通过 CDP 连接到 Windows PC（100.97.242.124:19223）的快手创作者平台
- 成功填写文案并上传图片（最多 9 张）
- 点击发布并确认成功
- 错误处理完整（连接失败、发布失败等）

**非目标**:
- 不实现 OAuth2 开放平台 API（无 App Key/Secret 凭据）
- 不处理视频发布（仅图文）
- 不实现定时发布功能

**参考实现**: `workflows/platform-data/workflows/publisher/scripts/publish-weibo.cjs`

### 内容 JSON 格式

```json
{
  "type": "kuaishou",
  "id": "unique-id",
  "content": "快手图文文案（必填）",
  "images": [
    "/path/to/image1.jpg"
  ]
}
```

### 架构

```
Mac mini → CDP → Windows PC (100.97.242.124:19223) → cp.kuaishou.com
```

### 异常处理 & 报警规范

| 异常场景 | 处理方式 | 报警级别 | eventType |
|----------|----------|----------|-----------|
| CDP 连接失败 | 输出错误日志，exit 1 | P2 | kuaishou_cdp_connect_failed |
| 未找到快手标签页 | 输出错误，提示用户打开 cp.kuaishou.com | P2 | kuaishou_tab_not_found |
| 文案填写失败 | 重试一次后 exit 1 | P2 | kuaishou_content_fill_failed |
| 图片上传失败 | 跳过该图片，继续发布 | P3 | kuaishou_image_upload_failed |
| 发布按钮未找到 | exit 1 | P2 | kuaishou_publish_btn_not_found |
| 发布超时 | 假定成功（降级处理） | P3 | kuaishou_publish_timeout |
