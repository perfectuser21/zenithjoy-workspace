# PRD: 作品详情页 (Page View + TipTap Editor)

## 📋 任务概述

这是 **Phase 3: 作品详情** 的实现。API 和列表页已完成（Phase 1-2），现在需要实现作品详情页和富文本编辑器。

**目标**：创建类似 Notion Page View 的作品详情页面，支持富文本编辑、媒体上传和自定义字段。

## 🎯 核心功能

### 1. 作品详情页面

**路由**: `/works/:id`

**功能点**：
- 显示作品完整信息（标题、类型、状态、日期等）
- 富文本内容编辑器（TipTap）
- 媒体文件展示（图片/视频）
- 自定义字段渲染
- 保存/更新按钮
- 返回列表按钮

### 2. TipTap 富文本编辑器

**集成**：
- 安装 @tiptap/react、@tiptap/starter-kit
- 基础文本格式（粗体、斜体、标题、列表等）
- 图片插入
- 代码块
- 链接

**UI 要求**：
- 工具栏（格式化按钮）
- 实时预览
- 自动保存提示

### 3. 媒体上传功能

**图片上传**：
- 点击上传或拖拽
- 预览缩略图
- 支持多图上传
- 存储到 media_files 数组

**视频上传**：
- 支持视频文件
- 显示视频播放器
- 存储到 media_files 数组

### 4. 自定义字段渲染

**字段类型**（Phase 3 支持基础类型）：
- 文本字段
- 数字字段
- 日期字段
- 下拉选择

**UI**：
- 字段名称标签
- 对应的输入组件
- 保存到 custom_fields JSONB

### 5. 保存/更新功能

**自动保存**：
- 编辑后 2 秒自动保存
- 显示保存状态（"保存中..." / "已保存"）

**手动保存**：
- Ctrl+S 快捷键
- 顶部"保存"按钮

## 📐 技术架构

### 前端技术栈

- **富文本编辑器**: TipTap (基于 ProseMirror)
- **UI 库**: Tailwind CSS
- **状态管理**: React Query (数据同步) + React State (编辑器状态)
- **路由**: React Router v6
- **HTTP 客户端**: Axios (已有 works.api.ts)
