# PRD: NotebookLM 图片生成工具

## 功能描述

通过 Playwright 自动化操作 NotebookLM，利用会员权益中的 Nano Banana 生成图片。

- 登录状态管理（保存/加载 Google 登录）
- 自动生成图片（Infographic/Slide）
- 下载图片到本地

## 成功标准

- [ ] scripts/notebooklm-login.py 创建（保存登录状态）
- [ ] scripts/notebooklm-image.py 创建（生成图片）
- [ ] 能成功生成并下载图片
