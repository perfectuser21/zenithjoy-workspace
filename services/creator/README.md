# ZenithJoy Creator

Content Creation System - Automated content generation and publishing platform.

![CI](https://github.com/perfectuser21/zenithjoy-creator/actions/workflows/ci.yml/badge.svg)

## Overview
This repository contains the ZenithJoy content creation system, providing automated content generation, card creation, and multi-platform publishing capabilities.

## Features
- 🎨 Automated luxury card generation
- 📝 Content management with SQLite/FastAPI
- 🚀 Multi-platform publishing (Toutiao, etc.)
- 🤖 AI-powered content generation
- 📊 Content library with 164+ pieces
