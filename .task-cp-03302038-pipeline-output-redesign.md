# Task Card: PipelineOutputPage UI 重设计

## 需求（What & Why）
**功能描述**: 重设计 `apps/dashboard/src/pages/PipelineOutputPage.tsx`，对齐 WorksGalleryPage 的视觉风格，修复已知显示问题，提升内容产出页面的信息密度和视觉质量。

**背景**: 当前 PipelineOutputPage 使用 gray/indigo 色调，与 WorksGalleryPage（slate/violet 调色板 + dark mode）的视觉风格不一致。另外：阶段耗时全部显示为 "0s"（因任务批量插入导致时间戳相差 <1ms）；没有展示 content-output 导出目录路径；没有展示 `content_publish` 阶段的状态。

**不做什么**: 不修改 Brain API 后端；不改路由配置；不做跨平台作品聚合（future work）。

## 成功标准

1. [ARTIFACT] `PipelineOutputPage.tsx` 使用 slate/violet 颜色方案，与 WorksGalleryPage 视觉风格一致
2. [BEHAVIOR] 阶段耗时 < 1s 时显示 "< 1s" 而非 "0s"；无时间戳时显示 "—"
3. [BEHAVIOR] 页面右侧阶段面板展示 `content_publish` 阶段（如有），显示"发布"标签
4. [ARTIFACT] 页面顶部 hero 区展示封面图（如有 cover 类型图片）
5. [ARTIFACT] 右侧面板底部展示 content-output 文件导出路径信息
6. [GATE] CI 全部通过，TypeScript 无错误
