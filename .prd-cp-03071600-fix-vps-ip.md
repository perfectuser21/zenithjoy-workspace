# PRD: 修复 zenithjoy-workspace 中硬编码的老 VPS IP

## 背景

美国 VPS 已从旧节点（146.190.52.84）迁移到新旧金山 SFO3 节点（134.199.234.147）。
zenithjoy-workspace 仓库中多个脚本和 n8n workflow JSON 文件里硬编码了老 IP。
老 VPS 销毁后这些脚本会失败。

## 目标

将 zenithjoy-workspace 中所有活跃代码文件的老 VPS IP 替换为新 IP。

## 涉及文件

1. `workflows/platform-data/workflows/publisher/scripts/transfer-now.cjs`
2. `workflows/platform-data/workflows/publisher/scripts/file-receiver.cjs`
3. `workflows/platform-data/workflows/publisher/scripts/ensure-files-on-windows.cjs`
4. `workflows/platform-data/workflows/publisher/scripts/download-to-windows.cjs`
5. `workflows/platform-data/workflows/publisher/scripts/start-file-server.sh`
6. `workflows/n8n/media/*.json`（7 个 workflow 文件）

## 成功标准

- 仓库中不再出现 `146.190.52.84`（排除 .archived/、日志）
