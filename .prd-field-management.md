# PRD: 字段管理 (Field Management)

## 📋 任务概述

这是 **Phase 4: 字段管理** 的实现。Phase 1-3 已完成（数据库、API、列表页、详情页），现在需要实现类似 Notion 的字段管理功能。

**目标**：创建字段管理界面，允许用户动态增删改字段，灵活配置自定义字段。

## 🎯 核心功能

### 1. 字段管理页面

**路由**: `/works/fields`

**功能点**：
- 显示所有字段（核心字段 + 自定义字段）
- 新增自定义字段
- 编辑字段（名称、类型、选项）
- 删除字段（软删除，数据不丢失）
- 拖拽排序字段

### 2. 字段类型支持

| 类型 | 说明 | 存储方式 |
|------|------|----------|
| text | 单行文本 | string |
| textarea | 多行文本 | string |
| select | 单选下拉 | string + options[] |
| multiselect | 多选标签 | string[] + options[] |
| date | 日期选择器 | ISO date string |
| number | 数字输入 | number |
| checkbox | 复选框 | boolean |

### 3. 核心字段（不可删除）

以下字段固定存在，只能隐藏/显示：
- 标题 (title) - 文本
- 正文 (content_text) - 多行文本
- 内容类型 (content_type) - 单选
- 状态 (status) - 单选
- 账号 (account) - 单选
- 创建日期 (created_at) - 日期
- 更新日期 (updated_at) - 日期
- 发布日期 (first_published_at) - 日期

### 4. 字段定义表结构（已存在）

```sql
-- field_definitions 表（Phase 1 已创建）
id UUID PRIMARY KEY
field_name VARCHAR(100) UNIQUE NOT NULL
field_type VARCHAR(50) NOT NULL  -- text/textarea/select/multiselect/date/number/checkbox
display_label VARCHAR(200)
options JSONB  -- 单选/多选的选项列表
default_value TEXT
is_required BOOLEAN DEFAULT false
display_order INTEGER
is_visible BOOLEAN DEFAULT true
created_at TIMESTAMPTZ DEFAULT NOW()
updated_at TIMESTAMPTZ DEFAULT NOW()
```

**预设字段**（Phase 1 已插入）：
- tags (多选)
- priority (单选: 低/中/高)
- internal_notes (多行文本)
- target_audience (文本)

## 🔧 技术实现

### 字段 API 客户端

```typescript
// api/fields.api.ts
export interface FieldDefinition {
  id: string;
  field_name: string;
  field_type: 'text' | 'textarea' | 'select' | 'multiselect' | 'date' | 'number' | 'checkbox';
  display_label: string;
  options?: string[];
  default_value?: string;
  is_required: boolean;
  display_order: number;
  is_visible: boolean;
  created_at: string;
  updated_at: string;
}

export interface CreateFieldInput {
  field_name: string;
  field_type: string;
  display_label: string;
  options?: string[];
  default_value?: string;
  is_required?: boolean;
}

export async function getFields(): Promise<FieldDefinition[]>
export async function createField(field: CreateFieldInput): Promise<FieldDefinition>
export async function updateField(id: string, updates: Partial<FieldDefinition>): Promise<FieldDefinition>
export async function deleteField(id: string): Promise<void>
export async function reorderFields(ids: string[]): Promise<void>
```

### 字段管理 Hook

```typescript
// hooks/useFieldDefinitions.ts
export function useFieldDefinitions() {
  const queryClient = useQueryClient();

  const { data: fields, isLoading } = useQuery({
    queryKey: ['fields'],
    queryFn: getFields,
  });

  const createMutation = useMutation({
    mutationFn: createField,
    onSuccess: () => {
      queryClient.invalidateQueries({ queryKey: ['fields'] });
    },
  });

  const updateMutation = useMutation({
    mutationFn: ({ id, updates }) => updateField(id, updates),
    onSuccess: () => {
      queryClient.invalidateQueries({ queryKey: ['fields'] });
    },
  });

  const deleteMutation = useMutation({
    mutationFn: deleteField,
    onSuccess: () => {
      queryClient.invalidateQueries({ queryKey: ['fields'] });
    },
  });

  const reorderMutation = useMutation({
    mutationFn: reorderFields,
    onSuccess: () => {
      queryClient.invalidateQueries({ queryKey: ['fields'] });
    },
  });

  return {
    fields,
    isLoading,
    createField: createMutation.mutate,
    updateField: updateMutation.mutate,
    deleteField: deleteMutation.mutate,
    reorderFields: reorderMutation.mutate,
  };
}
```

### 拖拽排序实现

使用已安装的 `@hello-pangea/dnd`：

```typescript
// components/fields/FieldList.tsx
import { DragDropContext, Droppable, Draggable } from '@hello-pangea/dnd';

export function FieldList({ fields, onReorder }) {
  const handleDragEnd = (result) => {
    if (!result.destination) return;

    const items = Array.from(fields);
    const [reorderedItem] = items.splice(result.source.index, 1);
    items.splice(result.destination.index, 0, reorderedItem);

    // 更新 display_order 并调用 API
    const ids = items.map(item => item.id);
    onReorder(ids);
  };

  return (
    <DragDropContext onDragEnd={handleDragEnd}>
      <Droppable droppableId="fields">
        {(provided) => (
          <div {...provided.droppableProps} ref={provided.innerRef}>
            {fields.map((field, index) => (
              <Draggable key={field.id} draggableId={field.id} index={index}>
                {(provided) => (
                  <div
                    ref={provided.innerRef}
                    {...provided.draggableProps}
                    {...provided.dragHandleProps}
                  >
                    <FieldItem field={field} />
                  </div>
                )}
              </Draggable>
            ))}
            {provided.placeholder}
          </div>
        )}
      </Droppable>
    </DragDropContext>
  );
}
```
