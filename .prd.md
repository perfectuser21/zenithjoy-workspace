# PRD: Works Gallery 测试（边做边测 Phase 2-1）

## 背景

Phase 1 基础设施已完成（vitest + CI），现在开始"边做边测"工作流。

Works Gallery 是第一个已跑通的核心功能：
- ✅ 33 张 ChatGPT 风格卡片
- ✅ 标签筛选（认知/社交/效率）
- ✅ 详情面板（Notion 风格右侧滑出）
- ✅ 键盘导航（ESC 关闭，Arrow Up/Down 切换）

## 目标

为 Works Gallery 写 2-3 个核心测试，建立测试标准：
1. **显示测试** - 验证 33 张卡片正确渲染
2. **筛选测试** - 验证标签筛选功能
3. **交互测试** - 验证卡片点击和面板打开

## 测试范围

| 测试 | 覆盖功能 | 优先级 |
|------|----------|--------|
| 卡片渲染 | 显示 33 张，标题正确 | P0 |
| 标签筛选 | 点击标签，卡片数量变化 | P0 |
| 清除筛选 | 点击清除，恢复 33 张 | P1 |
| 详情面板 | 点击卡片，打开面板 | P1 |

## 不测试（此 PR 不涉及）

- 懒加载逻辑（需要 IntersectionObserver mock）
- 键盘导航（需要 keyboard event 测试）
- 图片加载（需要 mock image onLoad）

## 影响范围

- `apps/dashboard/src/pages/__tests__/WorksGalleryPage.test.tsx` - 新建测试文件
