# Platform Data API Backend

## 背景

前端 Platform Data 页面已实现，但缺少后端 API 服务导致所有请求返回 404。需要创建轻量级 API 服务连接 HK PostgreSQL TimescaleDB。

## 需求

### 功能需求

1. **API 端点**
   - `GET /api/media/platform-data/{platform}` - 获取指定平台数据
   - 支持平台：douyin, kuaishou, xiaohongshu, toutiao, weibo, zhihu, channels

2. **数据库连接**
   - 连接到 HK PostgreSQL (host.docker.internal:5432)
   - 数据库：timescaledb
   - 查询各平台的 raw_data 视图

3. **响应格式**
   ```json
   {
     "success": true,
     "platform": "douyin",
     "count": 100,
     "data": [...]
   }
   ```

### 技术需求

1. **技术栈**
   - Node.js + Express
   - pg (PostgreSQL client)
   - 轻量级，快速响应

2. **部署**
   - Docker 容器
   - 端口：5220（HK 本地）
   - nginx 反向代理 `/api/media/*` → `http://host.docker.internal:5220`

3. **环境变量**
   - `POSTGRES_HOST`（使用 host.docker.internal）
   - `POSTGRES_PORT` (5432)
   - `POSTGRES_DB` (timescaledb)
   - `POSTGRES_USER`
   - `POSTGRES_PASSWORD`

## 验收标准

- [ ] API 服务启动成功

- [ ] 前端页面显示数据（不再是 404）
