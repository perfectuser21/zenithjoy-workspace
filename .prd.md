# PRD: 作品列表前端 (Database View)

## 📋 任务概述

这是 **Phase 2: 作品列表** 的实现。API 已完成（Phase 1），现在需要实现前端 Database View。

**目标**：创建类似 Notion Database 的作品列表页面，支持筛选、排序、搜索和分页。

## 🎯 核心功能

### 1. 作品列表页面

**路由**: `/works`

**功能点**：
- 表格视图展示所有作品
- 显示字段：标题、类型、状态、发布日期
- 点击行打开作品详情（Phase 3）
- 响应式布局

### 2. 筛选功能

**筛选器**：
- 内容类型：全部 / 文本 / 图文 / 视频 / 长文 / 音频
- 状态：全部 / 草稿 / 待发布 / 已发布 / 已归档
- 账号：全部 / XXIP / XXAI

### 3. 排序和搜索

- 排序：按创建日期、更新日期、标题排序
- 搜索：标题搜索

### 4. 分页

- 每页显示：20 条（默认）
- 支持切换：10 / 20 / 50 / 100

### 5. 新建作品（简版）

- 点击 "+ 新建" 按钮
- 填写标题和类型（必填）
- 创建后跳转到详情页（Phase 3）

## 📐 技术架构

- **框架**: React 18
- **UI 库**: Ant Design / Tailwind CSS
- **状态管理**: React Query
- **路由**: React Router v6
- **HTTP 客户端**: Axios
