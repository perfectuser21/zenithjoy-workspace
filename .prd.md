# PRD: 修复今日头条发布功能 - 处理云端同步弹窗

## 背景

今日头条发布功能一直失败，经过完整测试和诊断，发现根本原因：

**每次点击"预览并发布"按钮后，都会弹出"已开启云端自动同步"提示框，必须先点击"我知道了"才能继续发布流程。**

当前 `vps_publisher.js` 的 `publishToutiao` 函数在点击"预览并发布"后，只处理了"确定"或"发布"按钮，**没有处理"我知道了"按钮**，导致发布流程被弹窗阻塞，永远无法完成。

## 问题定位

### 诊断过程

1. ✅ CDP 连接正常 - 能成功连接到浏览器（端口 19225）
2. ✅ 登录状态正常 - 已登录头条号
3. ✅ 内容填写成功 - 标题和正文都能正确填写
4. ✅ 按钮点击成功 - 能找到并点击"预览并发布"按钮
5. ❌ **弹窗阻塞** - 点击后弹出"云端同步"提示框，未处理

### 弹窗内容

```
已开启云端自动同步

1. 草稿将自动同步至云端，你可以在不同设备上编辑
2. 你可前往 设置 - 功能设置 - 内容隐私 更改该设置

[我知道了]
```

### 现有代码位置

文件：`workflows/platform-scraper/vps_publisher.js`
函数：`publishToutiao(cdp, task, files)`
问题行：第1982-1997行（处理确认弹窗的代码）

**现有代码只匹配**：
- `btn.textContent.includes('确定')`
- `btn.textContent.includes('发布')`

**缺少匹配**：
- `btn.textContent.includes('知道了')`

## 需求

### 功能需求

修改 `vps_publisher.js` 中的 `publishToutiao` 函数，在点击"预览并发布"后：

1. **新增步骤**：检测并处理"云端自动同步"弹窗
   - 查找包含"知道了"文本的按钮
   - 点击关闭弹窗
   - 等待弹窗消失

2. **保留步骤**：继续处理其他可能的确认弹窗（原有逻辑）

### 非功能需求

- **健壮性**：弹窗可能不出现（已关闭过），处理时不要报错
- **日志**：每个步骤输出日志，方便调试
- **等待时间**：给足够时间让弹窗出现和消失（2秒）

## 技术方案

### 修改位置

文件：`workflows/platform-scraper/vps_publisher.js`
行号：第1977-1997行（在"未找到发布按钮"检查后）

### 修改内容

将原来的步骤 9 拆分为两个步骤：

**新步骤 9**：处理"云端自动同步"弹窗
```javascript
// 9. 处理"云端自动同步"弹窗（必须先处理这个）
await sleep(2000);
log('头条', '处理云端同步弹窗...');
const popupResult = await cdp.send('Runtime.evaluate', {
  expression: `
    (function() {
      const buttons = document.querySelectorAll('button');
      for (const btn of buttons) {
        const text = btn.textContent.trim();
        if (text.includes('知道了') && !btn.disabled && btn.offsetWidth > 0) {
          btn.click();
          return 'closed_sync_popup: ' + text;
        }
      }
      return 'no_sync_popup';
    })()
  `
});
log('头条', `云端同步弹窗: ${popupResult.result.value}`);
await sleep(2000);
```

**新步骤 10**：处理其他确认弹窗（原有逻辑优化）
```javascript
// 10. 处理其他可能的确认弹窗
log('头条', '处理其他确认弹窗...');
await cdp.send('Runtime.evaluate', {
  expression: `
    (function() {
      const buttons = document.querySelectorAll('button');
      for (const btn of buttons) {
        const text = btn.textContent.trim();
        if ((text === '确定' || text === '发布' || text === '确认发布') && !btn.disabled && btn.offsetWidth > 0) {
          btn.click();
          return true;
        }
      }
      return false;
    })()
  `
});
```

**新步骤 11**：验证发布结果（原步骤 10，序号+1）
```javascript
// 11. 验证发布结果
log('头条', '验证发布结果...');
await sleep(5000);  // 增加等待时间，确保页面跳转完成
const verifyResult = await h.verifyPublishSuccess(cdp, 30000);
log('头条', `验证结果: ${verifyResult}`);
```

### 关键改进

1. **精确匹配"知道了"**：使用 `text.includes('知道了')` 专门处理云端同步弹窗
2. **增加可见性检查**：`btn.offsetWidth > 0` 确保按钮可见
3. **增加等待时间**：验证阶段从 3秒 → 5秒，确保页面跳转完成
4. **优化其他弹窗匹配**：精确匹配"确定"、"发布"、"确认发布"（用 `===`）
