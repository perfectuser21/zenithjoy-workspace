# PRD: 启用 AI 视频生成功能 Feature Flag

## 问题
用户反馈侧边栏看不到"AI 视频"菜单项，尽管代码已经部署。

## 根本原因
InstanceContext.tsx 的 autopilotConfig.features 对象缺少 `'ai-video-generation': true` feature flag。

navigation.config.ts 中定义的菜单项使用 `featureKey: 'ai-video-generation'`，但该 feature 未在 InstanceContext 中启用，导致 `filterNavGroups()` 过滤掉了该菜单项。

## 解决方案
在 `apps/dashboard/src/contexts/InstanceContext.tsx` 第 40 行（'platform-data' 后面）添加：

```typescript
'ai-video-generation': true, // AI 视频生成
```

## 影响范围
- 文件：1 个（InstanceContext.tsx）
- 改动：1 行
- 风险：低（仅添加 feature flag）

## 验收标准
1. 重新构建并部署后，侧边栏出现"AI 视频"菜单项
2. 点击菜单项可以正常进入 /ai-video 页面
3. 其他功能不受影响
