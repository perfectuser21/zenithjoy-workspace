# PRD: 修复 AI 视频生成 API 认证问题

## 问题描述
AI 视频生成功能返回 401 错误，原因是 ToAPI 的 API Key 没有正确配置。

## 根本原因
1. 环境变量 `VITE_TOAPIS_API_KEY` 未在 `.env.production` 中配置
2. 代码使用了错误的环境变量读取方式（`process.env` 而非 `import.meta.env`）

## 解决方案
1. 在 `.env.production` 添加 `VITE_TOAPIS_API_KEY`
2. 修改 `video-generation.api.ts` 中的 `getApiToken()` 函数，使用 `import.meta.env.VITE_TOAPIS_API_KEY`

## 验收标准
- 视频生成功能不再返回 401 错误
- 可以成功创建视频生成任务
