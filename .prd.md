# PRD: zenithjoy-dashboard 仓库初始化

## 背景

从 cecelia-workspace 分离 Autopilot Dashboard 前端，成为独立仓库。

## 目标

1. 清理 Cecelia/Core 依赖，保留纯 Autopilot 功能
2. 本地构建测试通过
3. 推送到 GitHub
4. 可从香港 VPS 拉取部署

## 范围

### 包含
- apps/dashboard/frontend 的所有代码
- vite.config.ts（清理 monorepo 路径）
- package.json（重命名为 @zenithjoy/dashboard）

### 不包含
- Core features（使用 stubs 替代）
- Cecelia 对话功能（保留代码但不启用）

## 验收标准

1. `npm install` 成功
2. `npm run build` 成功，无 TypeScript 错误
3. GitHub 仓库创建并推送
4. README 说明部署方式

## 技术决策

- 保留 CeceliaContext 代码（兼容性），但 Autopilot 实例不会触发
- @features/core 指向 src/stubs/core-features
