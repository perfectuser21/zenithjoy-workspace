# Task Card: 全面清理 N8N 相关代码和配置

## 需求（What & Why）
**功能描述**: 删除 dashboard 中所有 n8n 相关代码，包括 API 调用、webhook 触发、proxy 配置、菜单项和类型定义。
**背景**: 系统已不再使用 n8n，相关代码造成运行时错误（请求到不存在的 /api/n8n-webhook/ 端点），需要全量清除。
**不做什么**: 不迁移到替代方案，不删除 workflows/n8n/ 下的 JSON 文件（历史存档），不改 FeishuLogin 的扫码 UI。

## 要改的文件

| 文件 | 操作 |
|------|------|
| `scraping.api.ts` | 删除 n8n webhook 调用，triggerScrapingTask 返回"暂不可用" |
| `FeishuLogin.tsx` | 将 n8n webhook 调用替换为 `/api/feishu-login`（Mac API 预留接口） |
| `LoginPage.tsx` | 删除 N8N_WEBHOOK_BASE 和 n8n 调用，显示"功能维护中"提示 |
| `video-generation.api.ts` | uploadImage 函数改为调用 `/api/upload-video-frame` |
| `ExecutionStatus.tsx` | 删除 n8n-live-status API 调用，页面显示"暂无数据" |
| `ai-employees.api.ts` | fetchLiveStatus 改为返回空数据 |
| `vite.config.ts` | 删除两个 n8n proxy 规则 |
| `AdminSettingsPage.tsx` | 删除 N8n 工作流和 N8n 实时状态两个菜单项，删除 Workflow/Radio import |
| `vite-env.d.ts` | 删除 VITE_N8N_WEBHOOK_BASE 和 VITE_N8N_API_KEY 类型定义 |
