# Task Card: 内容工厂前端 — 创建入口 + Pipeline 状态展示

## 需求（What & Why）
**功能描述**:
1. ContentFactoryPage 新增"创建 Pipeline"表单（keyword + series 下拉 + notebook_id 可选 + angle 可选）
2. Pipeline 列表展示优化 — 显示子任务进度（4 阶段状态条）
3. 看板视图按正确的 pipeline status 分列（不是子任务 status）

**背景**: 当前前端只有只读看板，无法创建 pipeline。用户需要从 Dashboard 输入关键词发起内容生产流程。

**不做什么**:
- 不改 Brain 后端 orchestrator/executor（下一个 PR）
- 不做实际内容生成（那是 executor 的事）
- 不改 /share-card skill

## 成功标准
1. [BEHAVIOR] 用户在前端填写 keyword + 选择 series → 点击创建 → POST /api/brain/pipelines → 看板新增一条
2. [ARTIFACT] 创建表单包含 keyword（必填）、series 下拉（从 /api/brain/content-types 拉）、notebook_id（可选）、angle（可选）
3. [BEHAVIOR] Pipeline 卡片显示 4 阶段进度条（research → generate → review → export）
4. [GATE] 不破坏已有页面功能
