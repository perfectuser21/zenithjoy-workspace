# PRD: 作品列表前端 (Database View)

## 📋 任务概述

这是 **Phase 2: 作品列表** 的实现。API 已完成（Phase 1），现在需要实现前端 Database View。

**目标**：创建类似 Notion Database 的作品列表页面，支持筛选、排序、搜索和分页。

## 🎯 核心功能

### 1. 作品列表页面

**路由**: `/works`

**功能点**：
- ✅ 表格视图展示所有作品
- ✅ 显示字段：标题、类型、状态、发布日期、自定义字段
- ✅ 点击行打开作品详情（Phase 3）
- ✅ 响应式布局

### 2. 筛选功能

**筛选器**：
- 内容类型：全部 / 文本 / 图文 / 视频 / 长文 / 音频
- 状态：全部 / 草稿 / 待发布 / 已发布 / 已归档
- 账号：全部 / XXIP / XXAI

**UI 位置**：页面顶部 Tab 栏或 Dropdown

### 3. 排序功能

**可排序字段**：
- 创建日期（默认：最新优先）
- 更新日期
- 发布日期
- 标题（字母顺序）

**UI**: 列表表头点击排序，带排序方向指示（↑↓）

### 4. 搜索功能

**搜索范围**：
- 标题
- 正文内容（可选，后端支持）

**UI**: 顶部搜索框，实时搜索或按回车搜索

### 5. 分页功能

**分页参数**：
- 每页显示：20 条（默认）
- 支持切换：10 / 20 / 50 / 100

**UI**: 底部分页器，显示总数和当前页

### 6. 新建作品（简版）

**功能**：
- 点击 "+ 新建" 按钮
- 弹出对话框或跳转到新建页面
- 只需填写标题和类型（必填）
- 创建后跳转到详情页（Phase 3）

**字段**：
- 标题（必填）
- 内容类型（必填，下拉选择）
- 状态（默认：草稿）

## 🔧 API 集成

### 调用 Works API

```typescript
// hooks/useWorks.ts
import { useQuery } from '@tanstack/react-query';
import { getWorks } from '../api/works';

export function useWorks(params: {
  type?: string;
  status?: string;
  account?: string;
  search?: string;
  limit?: number;
  offset?: number;
  sort?: string;
  order?: 'asc' | 'desc';
}) {
  return useQuery({
    queryKey: ['works', params],
    queryFn: () => getWorks(params),
    keepPreviousData: true, // 分页时保持上一页数据
  });
}
```

### API 函数

```typescript
// api/works.ts
import axios from 'axios';
import { Work, ListResponse } from '../types/works';

const api = axios.create({
  baseURL: 'http://localhost:5200/api',
});

export async function getWorks(params: any): Promise<ListResponse<Work>> {
  const { data } = await api.get('/works', { params });
  return data;
}

export async function createWork(work: Partial<Work>): Promise<Work> {
  const { data } = await api.post('/works', work);
  return data;
}
```
